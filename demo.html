DEMO PROOF OF CONCEPT VERSION - USE AT YOUR OWN RISK!

We are still working on this. Please come back later.

This is a demonstration version of MotionInput (v2 - Sept 2021), an experimental build released for the public to try out and for testers to report logs from back to the MotionInput community (via the main website). It may not work on all hardware and has had limited testing done on various configurations of hardware.

Some features now and in the future may require administrator privileges to operate. Users should always refer to the main website at motioninput.com for further release information, bug fixes and access to the main community.
It is not intended for general purpose end-users and requires further development. Errors may occur, the system can crash, and warning messages may be shown. Some processes may not be calculated effectively/efficiently enough in this version. It is a proof of concept prototype and should be treated as such. No liabilities are given. Use at your own risk. You should always make sure you are operating in a safe space and with safety in mind of equipment and items, with enough room around you.

This version is compatible with Windows 10 and Windows 11 that have full access to running executable Windows x64 applications, with Microsoft Visual C++ Runtime installed. It includes hand tracking, touch point recognition, eye tracking, head movement tracking, exercise recognition with full body gestures, Kinect layer simulation for developers and two handed navigation in games. It allows you to control a number of Windows applications and games with a compatible webcam.
